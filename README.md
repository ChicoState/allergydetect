# Allergy Detect

## Frontend setup
* Install Android SDK and emulator as per [these](https://reactnative.dev/docs/set-up-your-environment) instructions
* If using MacOS install Homebrew if not already installed as per [these](https://brew.sh/) instructions 
* Install npx if not already installed (MacOS)
```bash
brew install npm
```
* Install Node modules (too large to be committed) 
```bash
cd fronted && npm install
```
* Start project
```bash
npx expo start
```
* If you're using vim and MacOS you may need to add `set regexpengine=0` to your `.vimrc`. [Link to issue](https://github.com/vim/vim/issues/17247)
* [More info](https://reactnative.dev/docs/environment-setup)
